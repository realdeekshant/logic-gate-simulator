# Logic Gate Simulator

A node based logic gate simulator made with python.

![Sample](/example.png)

- Requirement
  - Python 3
  - PyQt 5

- Nodes
  - Input
  - Output
  - OR Gate
  - AND Gate
  - NOR Gate
  - NAND Gate
  - XOR Gate
  - NXOR Gate
